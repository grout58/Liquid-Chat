## What's New — v1.2 (March 2026)

### Security
- **Keychain Password Storage** — IRC passwords are now stored securely in the macOS Keychain. Nothing sensitive is written to UserDefaults or disk in plaintext.

### Notifications & Awareness
- **Desktop Notifications** — Get notified when someone mentions your nickname or sends you a direct message, even when the app is in the background. Powered by `UNUserNotificationCenter`.
- **Unread Message Badges** — Channel sidebar now shows real unread counts. Mention badges appear in accent color; regular unread in muted style. Badges clear automatically when you switch to the channel.
- **Mention Highlighting** — Messages containing your nickname are highlighted with an accent background in the message list.

### Moderation
- **User Ignore List** — Right-click any user in the nicklist to ignore them. Messages, joins, parts, and quits from ignored users are silently filtered. Toggle off just as easily. Persists across sessions.

### Bouncer Support
- **ZNC Status Messages** — Messages from ZNC pseudo-users (`*status`, `*playback`, etc.) are now routed to dedicated sidebar channels instead of being silently discarded.

### Polish
- **Tab Completion Colon** — Nickname tab completion now always appends `: ` after the nick, consistent with IRC convention regardless of cursor position.

## IRC Protocol

### Connection Handshake

```
1. CAP LS 302                       // Request IRCv3 capabilities
2. PASS password (if needed)        // Server/bouncer auth
3. NICK nickname                    // Set nickname
4. USER username 0 * :realname      // User identification
5. CAP REQ :sasl server-time …     // Request capabilities
6. [SASL exchange]                  // PLAIN or EXTERNAL
7. CAP END                          // End capability negotiation
```

### Supported Commands

| Command | Description |
|---|---|
| `/join #channel` | Join a channel |
| `/part [reason]` | Leave current channel |
| `/msg nick message` | Send private message |
| `/me action` | Send action (`/me waves`) |
| `/whois nick` | Get user info (shown inline) |
| `/list` | Browse channels (handles 10,000+) |
| `/topic [text]` | View or set channel topic |
| `/kick nick [reason]` | Kick user |
| `/ban hostmask` | Ban user |
| `/quit [message]` | Disconnect |

**Automatic:** PING/PONG, CAP negotiation, BATCH delivery, server-time timestamps, auto-reconnect, channel rejoin.

## AI Features

### Smart Reply Suggestions
Quick reply chips appear automatically after a message arrives. Powered by Apple Intelligence on macOS 26+; falls back to keyword-based suggestions on earlier systems.

- Tap a chip to populate the input field
- 1.5s debounce — regenerates after each new message
- Configurable via Settings > Advanced > Enable Smart Replies

### Smart Channel Recommendations
Analyzes your recent conversation and suggests related channels with relevance scores and one-click join.

### Chat Summarization
Generates structured catch-up summaries:
- Key discussion points (3–5 bullets)
- Main topics and sentiment
- Action items and TODOs

### Configuration
**Settings > Advanced:**
- Enable AI Features (master toggle)
- Enable Smart Replies
- AI Temperature (0.0–1.0)
- Auto-Summarize Threshold

## Building and Running

### Requirements
- macOS 26.0 or later
- Xcode 26.3 or later
- Swift 6.0 or later
- Apple Intelligence enabled (for AI features; app works fully without it)

### Build Steps

```bash
git clone https://github.com/grout58/Liquid-Chat.git
cd Liquid-Chat
open "Liquid Chat.xcodeproj"
# Press ⌘R to build and run
```

### Quick Start

1. Launch Liquid Chat
2. Click **New Connection…** (⌘N)
3. Enter server details — e.g. `irc.libera.chat`, port `6697`, SSL enabled
4. Click **Connect**
5. Join a channel: `/join #swift`

**Keyboard shortcuts:**
- `⌘F` — Search messages
- `Tab` — Complete nickname in input field
- `↑ / ↓` — Navigate message history
